# Banking Management System

This is a simple Banking Management System project developed in Python.

## 📖 Project Description
This project allows users to perform basic banking operations such as:
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance

It is a console-based application written in Python.

## 🛠️ Technologies Used
- Python

## 🚀 How to Run the Project
1. Install Python on your system.
2. Download the project file.
3. Open terminal or command prompt.
4. Run the command:
   python Banking_management.py

## 📌 Features
- Simple and user-friendly interface
- Basic banking transactions
- Error handling for invalid inputs
